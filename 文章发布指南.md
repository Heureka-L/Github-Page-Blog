## 🎯 系统概述

本博客系统基于Jekyll构建，采用响应式设计，支持中英文双语切换，具备以下核心特性：

- **书籍式阅读体验**：文章采用书籍风格布局
- **智能导航系统**：顶部导航栏根据页面类型动态变化
- **内容管理系统**：支持书籍目录和文章分类管理
- **响应式设计**：适配桌面、平板、手机多种设备

## ✍️ 文章发布流程

### 1. 创建文章文件

在 `_posts/` 目录下创建新的Markdown文件，命名格式：
```
YYYY-MM-DD-文章标题.md
```

### 2. 文章头部配置

每篇文章必须包含YAML前置数据：

```yaml
---
layout: post        # 使用文章布局
title: "文章标题"
subtitle: "副标题"
date: YYYY-MM-DD HH:MM:SS
categories: [分类1, 分类2]
tags: [标签1, 标签2]
---
```

### 3. 内容编写规范

- 使用Markdown语法编写
- 支持数学公式（MathJax）
- 支持代码高亮
- 支持图片插入

## 📖 页面布局说明

### 文章阅读界面 (book.html)

**布局特点：**
- 顶部横幅设计，包含文章标题和作者信息
- 右侧固定导航栏：HOME、FOLLOW ME、CONTENT三个按钮
- 书籍式分页体验
- 响应式侧边栏

**导航按钮功能：**
- **HOME**：返回网站首页
- **FOLLOW ME**：跳转到关于页面
- **CONTENT**：跳转到书籍目录页面

### 书籍目录页面 (books.html)

**布局特点：**
- 黑色主题导航栏（仅在books.html页面生效）
- 网格布局展示所有书籍
- 每本书包含封面、标题、章节列表
- 响应式卡片设计

**导航栏样式：**
- 背景色：#1a1a1a（深黑色）
- 文字颜色：#e0e0e0（浅灰色）
- 悬停效果：#3498db（蓝色高亮）

## 🧭 导航设计规范

### 顶部导航栏结构

```html
<!-- 通用导航结构 -->
<nav class="navbar navbar-default navbar-custom">
    <div class="container-fluid">
        <div class="navbar-header">
            <a class="navbar-brand" href="{{ site.baseurl }}/">{{ site.title }}</a>
        </div>
        <div class="navbar-collapse">
            <ul class="nav navbar-nav navbar-right">
                <!-- 动态生成的页面链接 -->
            </ul>
        </div>
    </div>
</nav>
```

### 书籍页面特殊导航

在 `books.html` 页面中，导航栏会自动切换为黑色主题：

```css
/* 仅books.html页面的导航栏样式 */
.navbar-custom {
    background-color: #1a1a1a !important;
    border-color: #333 !important;
}

.navbar-custom .navbar-brand,
.navbar-custom .nav li a {
    color: #e0e0e0 !important;
}

.navbar-custom .nav li a:hover {
    color: #3498db !important;
}
```

## ⚡ 功能特性详解

### 1. 文章分类系统

- **技术文章**：存储在 `_posts/` 根目录
- **书籍章节**：通过 `_data/books.yml` 配置管理
- **隐藏文章**：存储在 `_posts/hidden/` 目录

### 2. 书籍管理系统

配置文件 `_data/books.yml` 结构：

```yaml
books:
  - title: "书籍标题"
    description: "书籍描述"
    chapters:
      - title: "章节标题"
        url: "/2025/08/18/章节链接"
      - title: "另一章节"
        url: "/2025/08/19/另一章节链接"
```

### 3. 响应式特性

- **桌面端**：完整导航栏 + 侧边栏
- **平板端**：折叠导航栏 + 优化布局
- **手机端**：汉堡菜单 + 单列布局

## 🎨 样式定制指南

### 导航按钮样式

当前文章页面的导航按钮使用统一样式：

```css
.book-nav-btn {
    padding: 6px 12px;
    border: 1px dashed #bdc3c7;
    border-radius: 20px;
    font-size: 14px;
    font-weight: 700;  /* 加粗文字 */

    text-decoration: none;
    transition: all 0.3s ease;
}

.book-nav-btn:hover {
    color: #3498db;
    border-color: #3498db;
}
```

### 颜色方案

- **主色调**：#3498db（蓝色）
- **辅助色**：#e0e0e0（浅灰）
- **背景色**：#1a1a1a（黑色，书籍页面专用）
- **边框色**：#bdc3c7（中性灰）

## ❓ 常见问题解答

### Q1: 如何修改导航栏文字？

A: 修改对应页面的front matter中的`title`字段：

```yaml
---
title: "CONTENT"  # 这将更新导航栏显示
---
```

### Q2: 如何添加新的导航按钮？

A: 在 `_layouts/book.html` 中添加新的链接：

```html
<a href="{{ site.baseurl }}/新页面/" class="book-nav-btn">新按钮</a>
```

### Q3: 如何修改书籍目录页面样式？

A: 编辑 `books.html` 文件中的 `<style>` 标签，修改对应CSS规则。

### Q4: 文章发布后不显示？

A: 检查以下几点：
- 文件名格式是否正确（YYYY-MM-DD-标题.md）
- YAML前置数据是否完整
- 文章是否被正确提交到GitHub

### Q5: 如何设置文章特色图片？

A: 在文章头部添加：

```yaml
header-img: "img/post-bg-xxx.jpg"
```
